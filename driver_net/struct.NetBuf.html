Struct driver_net::NetBuf

source · 
pub struct NetBuf { /* private fields */ }
Expand description

A RAII network buffer.

It should be allocated from the NetBufPool, and it will be deallocated into the pool automatically when dropped.

The layout of the buffer is:

  ______________________ capacity ______________________
 /                                                      \
+------------------+------------------+------------------+
|      Header      |      Packet      |      Unused      |
+------------------+------------------+------------------+
|\__ header_len __/ \__ packet_len __/
|
